# Credits

## Upstream lineage

FnQuake3 builds on a long Quake III source lineage:

- [Quake III Arena SDK](https://github.com/id-Software/Quake-III-Arena) for the original open-sourced engine code.
- [ioquake3](https://github.com/ioquake/ioq3) for the long-running community-maintained continuation of the Quake III codebase.
- [Quake3e](https://github.com/ec-/Quake3e) for the modernized upstream foundation this repository is based on directly.

## Referenced and adapted work

FnQuake3 also draws on ideas, compatibility references, or adapted implementations from other projects:

- [CPMADevs CNQ3](https://bitbucket.org/CPMADevs/cnq3) for the rainbow text color escape support lineage.
- [Spearmint](https://github.com/zturtleman/spearmint) for the `bsp_q3ihv.c`-based Quake 3 IHV / `IBSP v43` map loading reference.
- [WolfcamQL](https://github.com/brugal/wolfcamql) for cross-checking legacy `.dm3` demo compatibility work.
- Luigi Auriemma's [qldec](https://aluigi.altervista.org/papers.htm#qldec) for the published Quake Live beta encrypted `.pk3` XOR table reference.
- [DarkMatter-Q2](https://github.com/Paril/DarkMatter-Q2) for cross-checking Quake II `.pak` and `.wal` compatibility support.

## Project thanks

Thanks to the broader Quake III community: players, server admins, mod authors, mappers, tool authors, testers, and maintainers who continue to keep the game playable and worth improving.
